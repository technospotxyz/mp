

MainActivity.java**

```java
package com.example.loginpage;

import androidx.appcompat.app.AppCompatActivity;
import android.content.Intent;
import android.os.Bundle;
import android.view.View;
import android.widget.Button;

public class MainActivity extends AppCompatActivity {
    Button btnGoToSecond;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        btnGoToSecond = findViewById(R.id.btnGoToSecond);

        btnGoToSecond.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                Intent intent = new Intent(MainActivity.this, SecondActivity.class);
                startActivity(intent);
            }
        });
    }
}
```

### üìù Steps of the Practical

1. **Create a new Android Project**

   * Open **Android Studio** ‚Üí **New Project** ‚Üí Select **Empty Activity** ‚Üí Name it `LoginPage`.

2. **Setup MainActivity and SecondActivity**

   * `MainActivity` is created automatically.
   * Create a new Java class `SecondActivity.java`.

3. **Design UI Layouts**

   * Open `res/layout/activity_main.xml` ‚Üí paste the fixed XML.
   * Create another layout `activity_main2.xml` ‚Üí paste the second XML.

4. **Connect Button to Second Activity**

   * In `MainActivity.java`, we added an `Intent` inside `btnGoToSecond.setOnClickListener()` to switch pages.

5. **Declare SecondActivity in Manifest**
   Open `AndroidManifest.xml` and **add SecondActivity**:

   ```xml
   <activity android:name=".SecondActivity"></activity>
   ```
